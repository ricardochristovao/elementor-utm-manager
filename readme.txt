== Description ==

O Elementor UTM Manager é uma solução completa para gerenciar parâmetros UTM em seus botões do Elementor.

Características principais:

* Suporte a todos os parâmetros UTM (source, medium, campaign, term, content)
* Cópia automática de queries do domínio pai
* Interface intuitiva integrada ao Elementor
* Compatível com as últimas versões do WordPress e Elementor

== Installation ==

1. Faça upload dos arquivos do plugin para o diretório `/wp-content/plugins/`
2. Ative o plugin através do menu 'Plugins' no WordPress
3. Use o Elementor para começar a adicionar UTMs aos seus botões

== Frequently Asked Questions ==

= O plugin é compatível com o último Elementor? =

Sim, o plugin é mantido atualizado com as últimas versões do Elementor.

= Como adicionar UTMs a um botão? =

Edite qualquer botão no Elementor e procure pela seção "UTM Settings".
